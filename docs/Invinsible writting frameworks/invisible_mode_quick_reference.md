# Invisible Mode Repurposing: Quick Reference Card

## THE 7 MODULES (Tag Your Long-Form With These)

| Tag | Module | What to Extract | Extraction Cue |
|-----|--------|-----------------|----------------|
| [M1] | Destabilize | Contradictions, paradoxes | "Most people think X, but..." |
| [M2] | Expose | Hidden scripts named | "You've been trained to..." |
| [M3] | Pressure | Costs (stated calmly) | "By doing nothing, you..." |
| [M4] | Reframe | Higher lens offered | "The real question is..." |
| [M5] | Identity | Self-concept challenged | "Very few people..." |
| [M6] | Agency | Responsibility transferred | "You must decide..." |
| [M7] | Exit | Tension left open | Questions without answers |

---

## SINGLE POST COMBINATIONS

```
COMBO A: M1 + M4 + M7
Hook challenges → Reframe offered → Loop stays open

COMBO B: M2 + M3 + M6  
Script named → Cost shown → Agency returned

COMBO C: M5 + M2 + M7
Identity challenged → Script exposed → No closure
```

## CAROUSEL SLIDE MAP

```
SLIDE 1 → M1 (Hook)         5-15 words
SLIDE 2 → M2 (Expose)       30-60 words
SLIDE 3 → M3 (Pressure)     30-50 words
SLIDE 4 → M4 (Reframe)      40-80 words
SLIDE 5 → M4/M2 (Path)      50-80 words
SLIDE 6 → M5+M6 (Identity)  40-70 words
SLIDE 7 → M7 (Open Exit)    15-30 words
```

---

## EXTRACTION CHECKLIST

Before posting, verify each module:

- [ ] **M1:** Does it interrupt a default frame?
- [ ] **M2:** Does it name an invisible script (no accusation)?
- [ ] **M3:** Is cost stated as inevitable, not threatening?
- [ ] **M4:** Does old question dissolve?
- [ ] **M5:** Does it describe who they want to be?
- [ ] **M6:** Is there NO rescue (no step-by-step)?
- [ ] **M7:** Is there NO tidy conclusion?
