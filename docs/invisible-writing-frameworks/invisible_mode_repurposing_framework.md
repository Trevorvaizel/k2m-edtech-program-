# The Invisible Mode Repurposing Framework
## From Long-Form Essays to High-Impact Social Posts

---

## Part 1: The Core Translation Logic

### What Changes Between Formats

| Long-Form | Social Media |
|-----------|--------------|
| All 7 modules in sequence | 2-3 modules per post |
| 1,500-3,000 words | 50-300 words per post |
| Single complete loop | Distributed loop across feed |
| Reader commits upfront | Must earn attention per scroll |
| Closure available at end | Closure denied per post |
| Identity pressure builds slowly | Identity pressure hits immediately |

### The Fundamental Principle

**Long-form = One deep cut**
**Social media = Death by a thousand paper cuts**

Each post must be complete enough to stand alone, yet incomplete enough to create hunger for more.

## Part 2: The Extraction Process

### Step 1: Module Mining

Read your long-form piece and highlight/tag every section by module:

```
[M1] = Destabilize (frame breaks, contradictions, pattern interrupts)
[M2] = Expose (default logic named, invisible scripts surfaced)
[M3] = Pressure (costs shown without drama)
[M4] = Reframe (higher lens offered, question dissolved)
[M5] = Identity (self-concept challenged or affirmed)
[M6] = Agency (responsibility transferred, no rescue)
[M7] = Exit (tension left unresolved)
```

### Step 2: Extract Standalone Fragments

For each tagged section, ask:
- Can this stand alone as a complete thought?
- Does it create a micro-loop (tension → partial resolution)?
- Would someone share this without context?

### Step 3: Identify Your "Payload Sentences"

These are the lines that carry maximum psychological weight per word. Look for:
- Paradoxes ("If you need to memorize it, you don't understand it")
- Reversals ("The best way to learn skills is to stop trying to learn skills")
- Identity statements ("Very few people have the courage")
- Specificity that signals depth ("85% of AI implementations fail")
- Metaphors that reframe ("Your mind is the digestive system for reality")

## Part 3: Post Type Architecture

### Type A: The Single-Slide Punch (Text Only)

**Structure:**
```
[Hook - M1 or M5] 1 sentence
[Body - M2 or M4] 2-4 sentences  
[Exit - M7 or M6] 1 sentence (no closure)
```

**Best Module Combinations:**
- M1 + M4 + M7 (Destabilize → Reframe → Open Exit)
- M2 + M3 + M6 (Expose → Pressure → Agency)
- M5 + M2 + M7 (Identity → Expose → Open Exit)

**Example from Dan Koe (Image 7):**
```
[M1] When everyone has an advantage, it is no longer an advantage.

[M4] When everyone can learn and create anything at the click of a 
button, your advantage comes from slowing down, focusing on your 
craft, doing the right things manually, and acquiring knowledge 
that is so specific to you that nobody can generate it with AI.

[M7] (Implicit: What knowledge is specific to YOU?)
```

**Word Count:** 50-150 words
**Use When:** You have a single powerful reframe or identity challenge

---

### Type B: The Carousel (Multi-Slide)

This is where the full module sequence lives—distributed across slides.

**Standard Carousel Architecture (5-7 slides):**

```
SLIDE 1: HOOK (M1)
├── Pattern interrupt
├── Contradiction or provocative claim
├── Visual: Bold text, minimal design
└── Goal: Stop the scroll

SLIDE 2: EXPOSE (M2)
├── Name the invisible script
├── "Most people think..." / "You've been told..."
├── Visual: Text only, clean
└── Goal: Recognition ("that's me")

SLIDE 3: PRESSURE (M3)
├── Show the cost (no drama)
├── "By doing nothing, you don't stay the same..."
├── Visual: Text only or metaphor image
└── Goal: Discomfort with status quo

SLIDE 4: REFRAME (M4)
├── Offer higher lens
├── "The question isn't X, it's Y"
├── Visual: Text only, emphasis on key phrase
└── Goal: Old question dissolves

SLIDE 5: IDENTITY (M5)
├── Challenge or affirm self-concept
├── "Very few people..." / "This is what it means to..."
├── Visual: Text or aspirational figure image
└── Goal: "I want to be that person"

SLIDE 6: AGENCY (M6)
├── Hand back responsibility
├── One clear decision point (not a system)
├── Visual: Text only
└── Goal: Weight stays with reader

SLIDE 7: OPEN EXIT (M7)
├── Question without answer OR
├── Statement that points beyond
├── NO summary, NO CTA to close loop
└── Goal: They keep thinking after swipe
```

---

### Type C: The Image-Text Hybrid

**Structure:**
```
IMAGE: Visual metaphor (solitary figure, path, light/dark contrast)
TEXT OVERLAY: [M1] or [M5] - Maximum 15 words
CAPTION: [M2] + [M6] - Expand and transfer agency
```

**Dan Koe's Visual Language:**
- Black and white illustrations
- Single figure in contemplative pose
- Vast space (void, sky, path ahead)
- Light source the figure faces or sits in

**Psychological Function of Visuals:**
- Reader projects themselves into the solitary figure
- The journey/path imagery encodes "this is a decision point"
- Light/dark contrast = current state vs. possible state

## Part 4: Module-by-Module Extraction Templates

### Extracting M1 (Destabilize) Hooks

**Source Material Indicators:**
- "Most people think X, but actually..."
- "The conventional wisdom says..."
- "Here's what nobody tells you about..."
- Any paradox or counterintuitive claim

**Extraction Formula:**
```
LONG-FORM: "Most organizations focus on AI tools first, then wonder 
why adoption fails. The conventional implementation playbook has it 
backwards—85% of AI projects fail because we're asking the wrong 
question entirely."

EXTRACTED HOOK: "85% of AI implementations fail. Not because of 
bad technology. Because of unprepared humans."

OR

EXTRACTED HOOK: "Don't try to implement the 'best AI tools.' 
Build the humans first."
```

**Quality Check:**
- [ ] Does it contradict something the reader assumes?
- [ ] Is it specific enough to signal depth?
- [ ] Can they NOT scroll past without reacting?

---

### Extracting M2 (Expose) Body Copy

**Source Material Indicators:**
- Descriptions of common behaviors/thoughts
- "The default approach assumes..."
- "We've been trained to..."
- Named patterns or invisible systems

**Extraction Formula:**
```
LONG-FORM: "Leaders who don't understand how to think strategically 
with AI, teams who haven't developed the cognitive capabilities to 
work alongside intelligent systems, organizations that try to bolt 
AI onto unchanged human processes—this is the consciousness gap in 
AI implementation."

EXTRACTED EXPOSE: "You've been taught to ask 'which tools?' before 
asking 'which humans?' That's the gap. The technology isn't the 
problem. The consciousness is."
```

**Quality Check:**
- [ ] Does it name something the reader does unconsciously?
- [ ] Is the tone observational, not accusatory?
- [ ] Will they think "I've been doing this"?

---

### Extracting M3 (Pressure) Stakes

**Source Material Indicators:**
- Consequences described
- "If you continue to... then..."
- Cost/benefit analysis
- Entropy or decay descriptions

**Extraction Formula:**
```
LONG-FORM: "Every month your organization delays human development, 
you're training teams to resist the very systems you'll need them 
to embrace. The gap between your AI strategy decks and what's 
actually changing on the ground widens. This isn't a future risk—
it's already happening."

EXTRACTED PRESSURE: "Every month you delay, you're training your 
team to resist what's coming. By doing nothing, you don't stay the 
same. You fall behind."
```

**Quality Check:**
- [ ] Is the cost stated as inevitable, not threatening?
- [ ] Does it feel like a diagnosis, not a warning?
- [ ] Is it free of melodrama or catastrophizing?

---

### Extracting M4 (Reframe) Lens Shifts

**Source Material Indicators:**
- "The real question is..."
- "What if instead of X, we considered Y?"
- Framework introductions
- Level shifts (from tactic to strategy, from doing to being)

**Extraction Formula:**
```
LONG-FORM: "The question isn't 'which AI tools should we implement?' 
The question is 'which humans are ready to think alongside intelligent 
systems?' That reframe changes everything about where you start, what 
you measure, and how you define success."

EXTRACTED REFRAME: "The question isn't 'which AI tools?' 

The question is 'which humans are ready?'

That changes everything."
```

**Quality Check:**
- [ ] Does the old question dissolve (not just get answered)?
- [ ] Is the new lens visibly higher-order?
- [ ] Can they not go back to the old frame?

---

### Extracting M5 (Identity) Challenges

**Source Material Indicators:**
- "The kind of person who..."
- Descriptions of admirable traits/behaviors
- "Very few people..."
- "This is what separates..."

**Extraction Formula:**
```
LONG-FORM: "Organizations that understand this—that develop their 
people while deploying their tools—will have an unfair advantage. 
This requires leaders with the courage to invest in human development 
before they can show AI ROI. Very few executives have this patience."

EXTRACTED IDENTITY: "The leaders who win the AI age won't be the 
ones with the best tools.

They'll be the ones with the patience to develop humans first.

Very few executives have this courage."
```

**Quality Check:**
- [ ] Does it describe who the reader wants to be?
- [ ] Is there implicit pressure (are you one of the few)?
- [ ] Does current behavior conflict with aspired identity?

---

### Extracting M6 (Agency) Transfers

**Source Material Indicators:**
- "You must decide..."
- "The choice is yours..."
- Action items framed as decisions, not instructions
- Consequences of choosing vs. not choosing

**Extraction Formula:**
```
LONG-FORM: "You can address this now, with your own development and 
your leadership team's. Or you can wait until adoption failure forces 
the conversation. Either way, the question will arrive. You're 
choosing when—and at what cost."

EXTRACTED AGENCY: "You can start now. Or you can wait until failure 
forces the question.

Either way, the question arrives.

You're just choosing the cost."
```

**Quality Check:**
- [ ] Is responsibility clearly with the reader?
- [ ] Is there NO rescue (no step-by-step system given)?
- [ ] Does the weight stay with them after reading?

---

### Extracting M7 (Exit) Open Loops

**Source Material Indicators:**
- Ending questions in your long-form
- Implications left unstated
- "What would it look like if..."
- Forward-pointing statements

**Extraction Formula:**
```
LONG-FORM: "The organizations that will win in the AI age aren't 
the ones with the best tools. They're the ones with the most 
conscious humans using those tools. What does that require of you—
starting tomorrow morning?"

EXTRACTED EXIT: "The winners won't have the best tools.

They'll have the most conscious humans.

What does that require of you?"
```

**Quality Check:**
- [ ] Is there NO tidy conclusion?
- [ ] Will they keep thinking about this later?
- [ ] Does it point toward action without specifying it?

### The 5-Slide Compressed Carousel

For medium-depth extractions:

```
SLIDE 1: Hook (M1)
SLIDE 2: Expose + Pressure (M2 + M3)
SLIDE 3: Reframe (M4)
SLIDE 4: Identity + Agency (M5 + M6)
SLIDE 5: Open Exit (M7)
```

### The 3-Slide Minimum Viable Carousel

For quick hits:

```
SLIDE 1: Hook (M1) + Expose (M2)
SLIDE 2: Reframe (M4) + Identity (M5)
SLIDE 3: Agency (M6) + Open Exit (M7)
```

## Part 6: The Repurposing Workflow

### From One Long-Form Piece, Extract:

```
1x Long-form essay (1,500-3,000 words)
                    ↓
         ┌─────────┴─────────┐
         ↓                   ↓
   3-5 Single Posts    1-2 Carousels
   (Type A)            (Type B)
         ↓                   ↓
   [M1+M4+M7]          [Full 7-slide]
   [M2+M3+M6]          [Compressed 5-slide]
   [M5+M2+M7]
         ↓                   ↓
         └─────────┬─────────┘
                   ↓
            1x Image Post
            (Type C)
                   ↓
     [Visual metaphor + M5 overlay + M2 caption]
```

### Step-by-Step Process

**STEP 1: READ & TAG (15 minutes)**
- Read your long-form piece
- Tag every section with module codes [M1] through [M7]
- Highlight "payload sentences" (maximum impact per word)

**STEP 2: EXTRACT HOOKS (10 minutes)**
- Pull all [M1] fragments
- Refine into 5-15 word hooks
- Aim for 3-5 usable hooks per long-form piece

**STEP 3: EXTRACT BODIES (15 minutes)**
- Pull [M2], [M3], [M4] fragments
- Pair them with hooks
- Build 2-3 single-post combinations

**STEP 4: EXTRACT IDENTITY/AGENCY (10 minutes)**
- Pull [M5] and [M6] fragments
- These often become carousel slides 5-6
- Or pair with strong hooks for identity-forward posts

**STEP 5: DESIGN CAROUSEL (20 minutes)**
- Select strongest hook
- Map full 7-module sequence across 5-7 slides
- Write slide-by-slide following blueprint

**STEP 6: CREATE IMAGE POST (10 minutes)**
- Identify the core metaphor in your piece
- Find or create visual that embodies it
- Write 15-word overlay + 50-word caption

## Part 7: Module Combination Cheat Sheet

### By Post Goal:

| If You Want To... | Lead With | Support With | Exit With |
|-------------------|-----------|--------------|-----------|
| Challenge assumptions | M1 | M4 | M7 |
| Create recognition | M2 | M3 | M6 |
| Inspire action | M5 | M6 | M7 |
| Shift perspective | M4 | M2 | M7 |
| Build urgency (without hype) | M3 | M5 | M6 |
| Create identity pull | M5 | M2 | M7 |

### By Audience State:

| If Your Audience Is... | Use This Sequence |
|------------------------|-------------------|
| Skeptical/resistant | M4 → M2 → M7 (soft infiltration) |
| Already problem-aware | M2 → M3 → M6 (pressure + agency) |
| Identity-driven | M5 → M2 → M7 (identity first) |
| Analytical/systems thinkers | M2 → M1 → M4 → M6 (systemic trap) |
| Ready to act | M1 → M5 → M6 → M7 (fast to agency) |

## Part 10: The Distribution Strategy

### Post Sequencing Across Your Feed

Just as Dan Koe's individual posts create micro-loops that accumulate across a scroll session, your posts should create a **distributed complete loop** across days/weeks.

**Weekly Distribution Pattern:**

```
MONDAY: M1-heavy post (Destabilize)
        → Break a frame your audience holds
        
WEDNESDAY: M2+M3 post (Expose + Pressure)  
           → Name the invisible system, show cost
           
FRIDAY: M4+M5 post (Reframe + Identity)
        → Offer new lens, challenge identity
        
SUNDAY: Carousel (Full M1-M7 sequence)
        → Complete loop for engaged followers
```

### The Accumulation Effect

Someone who sees all four posts gets the complete Invisible Mode treatment.

Someone who sees just one still gets a complete micro-loop.

Both are moved.
