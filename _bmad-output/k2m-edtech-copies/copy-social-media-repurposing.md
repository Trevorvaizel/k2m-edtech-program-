## Distribution Strategy

### Weekly Posting Pattern

**MONDAY:** M1-heavy post (break frame)
- "Everyone uses ChatGPT. But not everyone knows how."

**WEDNESDAY:** M2+M3 post (expose + pressure)
- "Most students copy-paste... habits follow you"

**FRIDAY:** M4+M5 post (reframe + identity)
- "The question isn't which prompts? It's do you understand what you're asking?"

**SUNDAY:** Full carousel (complete M1-M7 loop)

### Platform-Specific Tunings

**LinkedIn:** Professional tone, focus on career readiness
- Emphasize "think before prompt" as workplace skill
- Use "university" as proving ground

**Instagram/WhatsApp:** Peer aspiration, community
- "200 students learning together"
- Focus on "not being behind"
- Visual, story-driven

**Twitter/X:** Thread format, fast-paced
- Hook in first tweet
- Build tension through thread
- Drop revelation mid-thread
- Open question at end

## Quality Control Checklist

Before posting, verify:

**Hook (M1):**
- [ ] Does it interrupt default assumption about AI?
- [ ] Is it specific enough to signal depth?
- [ ] Would someone stop scrolling?

**Expose (M2):**
- [ ] Does it name invisible script ("copy-paste culture")?
- [ ] Is tone observational, not accusatory?
- [ ] Will they recognize themselves?

**Pressure (M3):**
- [ ] Is cost stated as inevitable, not threatening?
- [ ] Does it feel like diagnosis, not warning?

**Reframe (M4):**
- [ ] Does old question dissolve?
- [ ] Is new lens higher-order?

**Exit (M7):**
- [ ] Is there NO tidy conclusion?
- [ ] Will question echo after reading?

**Abstraction Check:**
- [ ] NO Layer 3 language ("AI literacy," "territory," "cartography")
- [ ] All concrete, immediate, relatable?
- [ ] 3-second comprehension test passed?
