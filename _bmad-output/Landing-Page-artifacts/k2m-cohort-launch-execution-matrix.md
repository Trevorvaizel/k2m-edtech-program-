## Daily Cadence (Minimum)

1. `08:00-09:00` Publish cold-touch status/posts (student + parent lanes).
2. `09:00-13:00` Connector and school outreach (new + follow-up).
3. `13:00-21:00` DM conversion window (Rungs 1-4), prioritize warm leads.
4. `21:00-21:30` Funnel log update: signals, warm leads, deposits, blockers.
