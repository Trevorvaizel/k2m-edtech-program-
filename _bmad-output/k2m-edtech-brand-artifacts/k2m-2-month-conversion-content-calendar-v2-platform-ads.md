## 1) High-Level Platform Reality (2026 Snapshot)

### Instagram (IG)
- Rewards recommendation eligibility + relevance to current viewer interests.
- Discovery depends on quality recommendations, not follower count alone.
- Reels ads are vertical-first, sound-forward, and skippable like organic reels.
- Private/social sharing and repeat utility behavior are strategic signals for distribution.

What IG uniquely rewards:
- Fast recognition in first seconds.
- Share/send-worthy and save-worthy practical value.
- Native-looking creative that feels like platform content, not repurposed TV.

### X (Twitter)
- Timeline ranking is built from explicit and implicit engagement signals (likes/replies + clicks/profile actions).
- Ranking stack uses candidate sourcing + rankers + visibility filters.
- Ad guidance supports broad initial targeting, then optimization with engagement-based remarketing.

What X uniquely rewards:
- Strong point of view.
- Reply-triggering claims.
- Thread structures that create bookmark value.

### LinkedIn
- Professional relevance and qualified intent matter more than viral entertainment.
- LinkedIn recommends avoiding over-layered targeting at launch.
- Lead gen objective and pre-filled forms can reduce friction.
- Accelerate/Auto-Targeting can optimize delivery from your signals and platform data.

What LinkedIn uniquely rewards:
- Practical frameworks.
- Operational clarity and outcomes.
- Credibility signals (data, process, implementation realism).

## 2) What Makes People Save

People save when the post is a reusable tool, not just a nice idea.

Use this SAVE test:
- S: Specific outcome ("use this before you submit AI-assisted work")
- A: Artifact (checklist, script, scorecard, rubric)
- V: Visual compression (one screen/slide can be screenshot)
- E: Execution now (usable in 2-5 minutes)

Persona save magnets:
- Students: "3-step self-check before you trust AI output"
- Parents: "5-question university readiness conversation script"
- Principals: "Pilot readiness scorecard for admissions season"

## 3) Paid Targeting Decision: General vs Student-First

Recommendation: Do not run generic "everyone" messaging first.

Best-performing structure for K2M:
- Student-primary creative system, plus parent and principal lanes from day 1.
- Broad delivery inside each lane (platform AI optimization), but persona-specific messaging.

Budget split (first 4 weeks):
- 55% Students (IG + X primary)
- 30% Parents (IG + LinkedIn)
- 15% Principals (LinkedIn primary)

Test lane:
- 10% of each lane reserved for broad lookalike/auto-expanded experiments.

Why:
- General messaging dilutes resonance and lowers conversion intent.
- Persona-specific creative gives stronger relevance signals to all three platforms.
- Platform guidance supports broader reach with fewer restrictions, but not vague messaging.

Use one 4-beat loop every week:

1. Recognition (pain seen clearly)

## 7) Paid Ad System: Nano Banana + Veo Workflow

Goal: Produce cinematic, emotional, platform-native ads ("big brand polish, direct response clarity").

### Stage A: Creative System Design
Define creative matrix:
- Personas: Student / Parent / Principal
- Funnel stage: Recognition / Capability / Commitment
- Emotion: Tension -> possibility -> agency

Create 9 core ad concepts (3 personas x 3 stages).

### Stage B: Frame Generation (Nano Banana / Gemini 2.5 Flash Image)
For each concept, generate:
- Character frame
- Environment frame
- End-state frame (desired outcome)

Prompt structure:
- Subject + setting + emotional state + lighting + lens style + brand palette

### Stage C: Motion Generation (Veo 3.1)
Use two primary modes:
- Ingredients to Video: maintain character/style consistency across shots.
- First and Last Frame: force narrative transitions with control.

Use this Veo prompt formula:
- [Cinematography] + [Subject] + [Action] + [Context] + [Style & Ambiance]

### Stage D: Ad Assembly
Build each ad in this sequence:
1. 0-2s Hook (pain recognition)
2. 2-8s Reframe (thinking vs copying)
3. 8-13s Proof (system/tool in action)
4. 13-15s CTA (apply/quiz/pilot)

Output set:
- 9:16 vertical master (IG/X primary)
- 1:1 square derivative (X/LinkedIn feed support)
- 16:9 version for LinkedIn video and site hero

### Stage E: Testing Protocol
For each audience lane, launch 3 variants:
- V1 Hook test
- V2 Proof test
- V3 CTA test

Kill/scale rule (every 72 hours):
- Kill if CTR < lane median and watch-through below benchmark.
- Scale winners by 20-30% budget increments.
