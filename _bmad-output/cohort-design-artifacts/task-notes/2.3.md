**Crisis Intervention Flow:**
1. Student posts message with crisis keyword in public channel
2. SafetyFilter.detect_crisis() immediately returns "mental_health_crisis"
3. Bot posts Kenya Crisis Response to channel: "💚 You matter. Hotline: 119, Trevor: 0116 405604"
4. notify_trevor_safety_violation() sends Trevor instant DM with student ID
5. Trevor reaches out within 1 hour (per Level 4 protocol)
6. If minor + high risk, Trevor calls parent within 2 hours

**Comparison Violation Flow:**
1. Student posts "top student" or similar comparison language
2. SafetyFilter.validate_no_comparison() raises ComparisonViolationError
3. Message blocked from posting
4. Trevor notified via DM with violation details
5. Logged to #moderation-logs

**Showcase Validation Flow:**
1. Student tries to share to #thinking-showcase with "draft" or "still working on"
2. SafetyFilter.validate_showcase_content() raises ComparisonViolationError
3. Message blocked with explanation: "#thinking-showcase is for finished work only"
4. Trevor notified of unfinished work attempt
