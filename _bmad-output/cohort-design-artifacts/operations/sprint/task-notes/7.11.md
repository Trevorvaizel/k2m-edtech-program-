## Apps Script Signing Code

Paste into `activateStudent()` in Apps Script before each webhook call:

```javascript
function _signedFetch(url, payload) {
  var payloadJson = JSON.stringify(payload);
  var secret = PropertiesService.getScriptProperties().getProperty('WEBHOOK_SECRET');
  var sig = Utilities.computeHmacSha256Signature(payloadJson, secret, Utilities.Charset.UTF_8);
  var sigHex = sig.map(function(b) {
    return ('0' + (b & 0xff).toString(16)).slice(-2);
  }).join('');
  return UrlFetchApp.fetch(url, {
    method: 'post',
    contentType: 'application/json',
    headers: { 'X-K2M-Signature': 'sha256=' + sigHex },
    payload: payloadJson,
    muteHttpExceptions: true
  });
}

// Store WEBHOOK_SECRET in Apps Script:
// Project Settings → Script properties → Add WEBHOOK_SECRET = <same value as Railway>
```

Usage:
```javascript
// Role upgrade
var result = _signedFetch(BOT_URL + '/api/internal/role-upgrade',
  { discord_id: discordId });

// Preload student
var result = _signedFetch(BOT_URL + '/api/internal/preload-student',
  { enrollment_email: email, enrollment_name: name, ... });
```

## Endpoint Specs

### POST /api/internal/role-upgrade

**Headers:** `X-K2M-Signature: sha256=<hmac>`

**Body:**
```json
{"discord_id": "123456789012345678"}
```

**Responses:**
- `200` — `{"success": true, "discord_id": "..."}`
- `400` — missing/non-numeric discord_id
- `401` — missing/invalid signature (logged to dashboard)
- `500` — Discord API error

### POST /api/internal/preload-student

**Headers:** `X-K2M-Signature: sha256=<hmac>`

**Body:**
```json
{
  "enrollment_email": "student@example.com",
  "enrollment_name": "First Last",
  "last_name": "Last",
  "invite_code": "abc123",
  "discord_id": "__pending__student@example.com",
  "cohort_id": "cohort-1",
  "start_date": "2026-04-01",
  "enrollment_status": "enrolled",
  "payment_status": "Confirmed"
}
```

**Responses:**
- `200` — `{"success": true}`
- `400` — invalid JSON or non-object body
- `401` — missing/invalid signature (logged to dashboard)
- `500` — DB error
